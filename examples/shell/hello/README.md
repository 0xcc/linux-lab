## Usage

    $ ls
    hello.sh
    $ cat hello.sh
    #!/bin/bash

    echo 'Hello, World!'

    $ chmod a+x hello.sh
    $ ./hello.sh
    Hello, World!

    $ bash hello.sh
    Hello, World!

    $ . hello.sh
    Hello, World!

    $ dash hello.sh
    Hello, World!

    $ echo 'Hello, World!'
    Hello, World!
